# ProtonHorizon Troubleshooting Guide

## Загальні Помилки

### Емулятор не запускається

**Помилка: "Emulator initialization failed"**
```
Рішення:
1. Перевірте наявність Vulkan драйверів: adb shell glxinfo | grep Vulkan
2. Переробіть JNI бібліотеки: ./gradlew clean assembleDebug
3. Перевірте версію Android API: мінімум API 24
```

**Помилка: "JNI library not found"**
```
Рішення:
1. Перебудуйте нативні компоненти: ./gradlew externalNativeBuildDebug
2. Перевірте наявність NDK: echo $ANDROID_NDK_ROOT
3. Видаліть кеш: rm -rf .gradle build/
```

### Проблеми з Геймпадом

**Геймпад не визнається**
```
Рішення:
1. Переведіть геймпад в режим впізнавання
2. Відключіть і переходіть Bluetooth
3. Перезавантажте додаток
4. Перевірте логи: adb logcat | grep Gamepad
```

**Геймпад працює неправильно**
```
Рішення:
1. Відкалібруйте геймпад у налаштуваннях
2. Оновіть прошивку геймпада
3. Перевірте чутливість аналогів
```

### Проблеми з Графікою

**Черний екран при запуску гри**
```
Рішення:
1. Спробуйте інший графічний API:
   - Спочатку Vulkan
   - Потім DXVK
   - Потім OpenGL
2. Оновіть GPU драйвери пристрою
3. Зменшіть роздільну здатність графіки
```

**Дуже низька FPS**
```
Рішення:
1. Зменшіть роздільну здатність
2. Отключіть VSync
3. Заприлегайте CPU частоту
4. Зменшіть якість текстур
```

**Артефакти в графіці**
```
Рішення:
1. Оновіть драйвери GPU
2. Спробуйте інший графічний API
3. Обновіть DXVK/VK3D версію
```

### Проблеми з Завантаженням Компонентів

**Помилка завантаження Proton/DXVK**
```
Рішення:
1. Перевірте інтернет-з'єднання
2. Перевірте місце на диску (потрібно 5GB+)
3. Спробуйте завантажити вручну і розмістити в:
   /sdcard/ProtonHorizon/components/
```

**Компонент не встановлюється**
```
Рішення:
1. Видаліть попередню версію
2. Перевірте права доступу
3. Перезавантажте додаток
4. Перевірте логи: logcat | grep Component
```

### Проблеми з Wine/Proton

**"Cannot start wine server"**
```
Рішення:
1. Очистіть Wine префікс: rm -rf ~/.wine
2. Переінстальйте Proton/Wine компоненти
3. Перевірте наявність необхідних бібліотек
```

**Гра крашується при запуску**
```
Рішення:
1. Перевірте совітимість гри (32/64-бітна)
2. Встановіть необхідні залежності (PhysX, DirectX Runtime)
3. Спробуйте інші версії Proton/Wine
4. Перевірте логи Wine: WINEDEBUG=+all wine game.exe
```

### Проблеми з Памʼяттю

**Додаток крашується з "Out of Memory"**
```
Рішення:
1. Закрийте інші додатки
2. Змашите кеш: Settings > Storage > Cached data
3. Зменшіть детальність графіки
4. Перезавантажте пристрій
```

## Просунуті Налаштування

### Логування

Дозволити детальне логування:
```bash
# Увімкнення всіх логів
adb shell setprop debug.atrace.tags.enableflags 0xffffffff

# Proton логування
WINEDEBUG=+all adb shell am start -n com.protonhorizon.emulator/.MainActivity

# DXVK логування
adb shell setprop persist.dxvk.loglevel info
```

### Оптимізація Батареї

```
1. Отключіть VSync для сохранення батареї
2. Зменшіть частоту оновлення дисплею
3. Включіть режим "Енергозбереження" у налаштуваннях
```
