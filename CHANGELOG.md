# CHANGELOG

## [1.0.0] - 2025-12-25

### Added
- **Основна емуляторна ядро** з підтримкою основних інструкцій
- **Підтримка геймпадів**:
  - Xbox контролери
  - PlayStation контролери
  - Nintendo Joy-Con
  - Steam контролери
  - Універсальні Bluetooth геймпади

- **Графічні API**:
  - Vulkan
  - DXVK (Direct3D 9-11 → Vulkan)
  - VK3D/VKD3D-Proton (Direct3D 12 → Vulkan)
  - OpenGL (базова підтримка)

- **Компоненти**:
  - Proton (GloriousEggroll-GE)
  - Wine (Stable, Staging)
  - DXVK (для D3D графіки)
  - VK3D-Proton (для D3D12 графіки)
  - Vulkan Loader і Layers

- **Завантаження Компонентів**:
  - Асинхронне завантаження з GitHub
  - Відновлення завантаження
  - Прогрес завантаження
  - Експорт/імпорт налаштувань

- **Оптимізація для Winlator**:
  - ARM64 оптимізація
  - Bionic GLIBC інтеграція
  - Мінімізація розміру
  - Оптимізація пам'яті

- **Інтерфейс**:
  - Головна активність з поверхнею для відображення
  - Управління налаштуваннями
  - Виділення компонентів
  - Монітор геймпада

### Technical
- C++17 для нативного кода
- JNI для зв'язку Java/C++
- CMake для побудови
- Gradle 8.1 для Android
- Kotlin для Android UI

### Performance
- Multi-threaded emulation
- GPU acceleration
- Memory management
- CPU frequency scaling support
